# Todo Evolution — Panaversity Hackathon 2026
**Spec-Driven Development with Spec-Kit Plus + Claude Code**

---

## Project Overview

A 3-phase todo application demonstrating the evolution from a simple console app to a full-stack web app to an AI-powered chatbot. Built using **Spec-Driven Development** with **Panaversity Spec-Kit Plus**.

---

## Methodology: Spec-Kit Plus

Every feature was built using the spec-driven workflow:

```
/sp.constitution  → .specify/memory/constitution.md
/sp.specify       → specs/<phase>/spec.md
/sp.plan          → specs/<phase>/plan.md
/sp.tasks         → specs/<phase>/tasks.md
/sp.implement     → actual code (Claude Code generated)
```

**Rule:** No code was written manually. All code was generated by Claude Code from the specs.

## Phase I — In-Memory Python Console App

**Tech:** Python, rich, pytest

### Run
```bash
cd phase-1-console
pip install -r requirements.txt
python main.py
```

### Features
- Add, delete, update, view, complete tasks
- Priority levels (high/medium/low) with color coding
- Tags/categories
- Search by keyword
- Filter by status, priority, tag
- Sort by priority, title, or status
- Due dates

### Test
```bash
pytest tests/ -v
```

## Phase II — Full-Stack Web Application

**Tech:** FastAPI, SQLModel, Neon DB (PostgreSQL), Next.js 14, Tailwind CSS

### Setup Backend
```bash
cd phase-2-web/backend
pip install -r requirements.txt
cp .env.example .env
# Add your Neon DB connection string to .env
uvicorn main:app --reload
# → http://localhost:8000
# → API docs: http://localhost:8000/docs
```

### Setup Frontend
```bash
cd phase-2-web/frontend
npm install
cp .env.local.example .env.local
npm run dev
# → http://localhost:3000
```

### Features
- All Phase I features + persistence
- Real-time UI updates
- Edit modal
- Color-coded priority badges
- Client-side search + server-side filtering
- Responsive dark UI

## Phase III — AI-Powered Todo Chatbot

**Tech:** OpenAI Agents SDK, MCP (Model Context Protocol), gpt-4o-mini

### Prerequisites
- Phase II backend must be running at localhost:8000

### Setup
```bash
cd phase-3-chatbot
pip install -r requirements.txt
cp .env.example .env
# Add your OPENAI_API_KEY to .env
python chatbot.py
```

### Example Conversations
```
You: Show me my pending tasks
You: Add buy groceries with high priority and tag home
You: Mark the groceries task as done
You: Reschedule my meeting to 2026-03-10
You: What's on my plate today?
You: Delete all completed tasks
```
